## Getting Started

Clone this repository:
```bash
git clone https://github.com/luizfverissimo/portfolio-2.git
```

Run:

```bash
yarn # or npm install
yarn dev # or npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
